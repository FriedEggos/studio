# **App Name**: JTMK+ Digital Hub

## Core Features:

- QR Code Program Registration: Enable students to register for JTMK programs by scanning a QR code, streamlining attendance tracking.
- Activity Evidence Upload and Verification: Allow students to upload activity evidence (images) and admins to verify the submissions.
- Automated Badge and E-Certificate Generation: Automatically generate badges and e-certificates upon successful activity verification using Cloud Functions.
- Real-Time Analytics Dashboard: Provide a real-time dashboard in Looker Studio to monitor program participation and KPIs, leveraging Firebase data via BigQuery.
- User Profile and Participation History: Enable students to view their profile and participation history, fostering engagement and providing a record of involvement.

## Style Guidelines:

- Primary color: Deep Blue (#1E3A8A) to represent professionalism and trust, aligning with the academic nature of JTMK. This color also evokes feelings of stability.
- Background color: Light Blue (#E3F2FD) to provide a clean, accessible, and modern feel.
- Accent color: Soft Lavender (#C5B3E6) to complement the primary blue and add a touch of innovation and creativity.
- Headline font: 'Space Grotesk' sans-serif font for titles and headings, complemented by 'Inter' sans-serif for body text.
- Utilize clean, modern icons that are intuitive and align with the digital theme of the application, such as QR code symbols and checkmark for verification.
- Implement a card-based layout to display program information, making it easy to scan and interact with different events.
- Use subtle animations, such as loading animations, to improve user experience and engagement, giving the user a good feedback about application activity.
